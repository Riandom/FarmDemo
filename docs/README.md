# Farm Demo（2D 像素农场模拟经营）

这是一个 Godot 4.5 的 2D 像素风种田模拟经营 Demo，目标是实现核心玩法闭环：**开垦 → 播种 → 浇水 → 生长 → 收获 → 售卖**。

## 环境要求

- Godot 版本：4.5
- 脚本语言：GDScript
- 游戏分辨率：800×600
- 贴图单位：32×32 像素

## 基本操作

- `W/A/S/D`：移动（对角线会做归一化，保持速度一致）
- `E`：交互（开垦 / 播种 / 浇水 / 收获，取决于地块状态）
- `I`：背包开关
- `B`：商店开关

## 核心玩法循环

1. 开垦：将荒地变为可耕作土地（翻土）
2. 播种：在耕作土地上种下作物
3. 浇水：给已播种地块浇水以启动生长计时
4. 生长：随时间推进作物分阶段成长
5. 收获：成熟后进行收获获得作物/收益
6. 售卖：在商店中将作物兑换为金币
