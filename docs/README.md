# Farm Demo（2D 像素农场模拟经营）

这是一个 Godot 4.5 的 2D 像素风种田模拟经营 Demo，当前目标已从单纯种田闭环扩展为：  
**种田经营 → 金币积累 → 世界区域切换 → 镇子/NPC 生活层 → 洞窟探索原型**。

## 环境要求

- Godot 版本：4.5
- 脚本语言：GDScript
- 游戏分辨率：1280×720（默认全屏）
- 贴图单位：32×32 像素

## 基本操作

- `W/A/S/D`：移动（对角线会做归一化，保持速度一致）
- `F`：交互（开垦 / 播种 / 浇水 / 收获 / 睡觉等）
- `I`：背包开关
- `B`：商店开关
- `O`：订单界面开关
- `Q / E`：热栏循环切换
- `1~0`：热栏快捷选择
- `Esc`：暂停菜单

## 核心玩法循环

1. 开垦：将荒地变为可耕作土地（翻土）
2. 播种：在耕作土地上种下多种作物
3. 浇水：给已播种地块浇水以启动生长计时
4. 生长：随时间推进作物分阶段成长，并受季节经营影响
5. 收获与售卖：将作物收获并出售，或用于订单目标
6. 经营准备：通过金币、背包、热栏、储物箱和订单系统组织日常经营
7. 生活层承载：玩家可在农场 / 小屋 / 镇子 / 洞窟之间切换，并与镇子 NPC 进行对话、送礼和委托互动
8. Phase 6 原型：已接入飞镖出征式战斗与单房间洞窟探索闭环，当前冻结在原型阶段
